# p5.js Museum Project

美術館のようなインタラクティブアート作品を展示するウェブサイトです。

## 🎨 プロジェクト概要

このプロジェクトは、p5.jsで作成されたインタラクティブアート作品を展示するためのウェブサイトです。美術館をイメージしたデザインで、訪問者は各作品を閲覧し、インタラクションを楽しむことができます。

## 📁 ディレクトリ構造

```
museum/
├── index.html        # 作品一覧ページ（ギャラリー）
├── css/
│   └── style.css     # スタイルシート
├── works/
│   ├── work1.html    # 作品1の展示ページ
│   ├── work2.html    # 作品2の展示ページ
│   └── work3.html    # 作品3の展示ページ
├── js/
│   ├── work1.js      # 作品1のp5.jsスケッチ
│   ├── work2.js      # 作品2のp5.jsスケッチ
│   └── work3.js      # 作品3のp5.jsスケッチ
└── lib/
    └── p5.min.js     # p5.jsライブラリ
```

## 🖼️ 展示作品

### 作品1: カラフルな円
- カラフルな円が画面上を動き回るインタラクティブな作品
- クリックすると新しい円が追加されます
- 円は壁にバウンドしながら動き続けます

### 作品2: 波紋のダンス
- マウスカーソルに追従する美しい波紋のアニメーション
- マウスを動かすと波紋が広がります
- ダークな背景に青い波紋が幻想的な雰囲気を作り出します

### 作品3: 幾何学の庭
- ランダムに生成される幾何学模様
- 三角形、四角形、六角形がカラフルに配置されます
- クリックすると新しいパターンが生成されます

## 🚀 使い方

1. ローカルサーバーを起動します：
   ```bash
   python3 -m http.server 8000
   # または
   npx serve
   ```

2. ブラウザで `http://localhost:8000` を開きます

3. ギャラリーページから各作品をクリックして閲覧します

## 🛠️ 技術スタック

- **HTML5** - ページ構造
- **CSS3** - スタイリング（グラデーション、アニメーション、レスポンシブデザイン）
- **JavaScript** - インタラクティブ機能
- **p5.js v1.7.0** - クリエイティブコーディングライブラリ

## 🎨 デザイン特徴

- 美術館をイメージした洗練されたUI
- 紫のグラデーションをテーマカラーに使用
- ホバーエフェクトで作品カードが浮き上がる
- レスポンシブデザイン対応
- 日本語対応

## 📝 カスタマイズ

新しい作品を追加する場合：

1. `works/` に新しいHTMLファイルを作成
2. `js/` に対応するp5.jsスケッチを作成
3. `index.html` のギャラリーに作品カードを追加
